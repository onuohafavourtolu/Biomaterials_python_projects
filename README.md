# Biomaterials_python_projects

These are beginner Python codes I built to apply concepts from my Wood and Biomaterials Engineering degree to real calculations. I am early in my coding journey but these projects reflect what I am learning and the problems I care about.


# Wood Moisture Content Calculator

This calculates the moisture content of wood samples using the standard engineering formula. You can either type in your values manually or upload a CSV file with multiple samples. I used West African wood species as my sample data because that is the data relevant to my work and interests.
